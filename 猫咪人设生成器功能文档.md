# 猫咪人设生成器功能文档

## 项目概述

猫咪人设生成器是一个基于AI的创意工具，通过分析猫咪的照片和特征信息，为每只猫咪生成独特的人设、背景故事和形象设计。

## 核心功能

### 1. 猫咪信息采集

#### 1.1 图片上传
- **功能描述**：用户可以上传猫咪的照片
- **支持格式**：JPG、PNG、WEBP
- **文件大小限制**：最大 5MB
- **上传方式**：
  - 点击上传区域选择文件
  - 拖拽文件到上传区域
- **预览功能**：上传后即时显示图片预览

#### 1.2 调查问卷

##### 必填项
- **猫咪姓名**：文本输入框
- **性别**：单选项
  - 公猫
  - 母猫
  - 未知
- **年龄**：下拉选择
  - 幼猫 (0-1岁)
  - 成猫 (1-7岁)
  - 老猫 (7岁以上)
- **品种**：下拉选择
  - 英国短毛猫
  - 美国短毛猫
  - 波斯猫
  - 暹罗猫
  - 布偶猫
  - 缅因猫
  - 中华田园猫
  - 其他

##### 选填项
- **性格特征**：多选复选框
  - 活泼
  - 安静
  - 粘人
  - 独立
  - 调皮
  - 温顺

- **喜好习惯**：多选复选框
  - 晒太阳
  - 玩玩具
  - 睡觉
  - 吃零食
  - 看窗外

#### 1.3 风格选择（必选）
- **写实风格**：真实感强，细节丰富
- **卡通风格**：可爱夸张，色彩明快
- **水彩风格**：柔和梦幻，艺术感强
- **油画风格**：古典优雅，质感厚重
- **赛博朋克**：未来科技，霓虹色调
- **古风风格**：中国传统，古典韵味
- **二次元风格**：动漫风格，萌系表现

### 2. AI人设生成

#### 2.1 生成内容
- **人设形象**：基于猫咪照片和特征生成的视觉形象
- **背景故事**：与猫咪性格、习惯强相关的创意故事
- **性格描述**：详细的人设性格特点
- **特殊能力/特长**：根据猫咪特征赋予的独特设定

#### 2.2 生成流程
1. 用户填写完整表单（包括上传图片、填写问卷、选择风格）
2. 点击"生成猫咪人设"按钮
3. 系统显示加载动画和提示文字
4. AI分析猫咪特征和用户选择的风格
5. 生成与猫咪信息强相关的人设内容
6. 展示生成结果

#### 2.3 关联性保证
生成的人设内容会充分考虑：
- 猫咪的品种特征
- 年龄阶段特点
- 性格特征
- 日常习惯
- 用户选择的艺术风格

## 技术架构

### 前端技术栈
- HTML5
- CSS3
- JavaScript (原生)

### 核心模块
- `index.html`：主页面结构
- `assets/css/main.css`：主样式文件
- `assets/js/main.js`：主逻辑控制
- `assets/js/upload.js`：图片上传处理
- `assets/js/dify-api.js`：AI接口调用
- `assets/js/utils.js`：工具函数
- `config/dify-config.js`：配置文件

### AI集成
- 使用 Dify API 进行AI内容生成
- 支持图片和文本混合输入
- 实时流式响应处理

## 用户交互流程

```
开始
  ↓
上传猫咪照片（必须）
  ↓
填写猫咪基本信息（必须）
  ↓
选择性格特征（可选）
  ↓
选择喜好习惯（可选）
  ↓
选择生成风格（必须）
  ↓
点击生成按钮
  ↓
AI分析处理
  ↓
展示生成结果
  ↓
结束
```

## 数据字段说明

### 表单数据结构
```javascript
{
  catName: String,        // 猫咪姓名
  gender: String,         // 性别：公猫/母猫/未知
  age: String,           // 年龄段：幼猫/成猫/老猫
  breed: String,         // 品种
  personality: Array,    // 性格特征数组
  habits: Array,         // 喜好习惯数组
  style: String,         // 生成风格
  image: File            // 猫咪照片文件
}
```

## 功能特点

1. **强关联性**：生成内容与猫咪实际特征高度相关
2. **多样化风格**：提供7种不同艺术风格选择
3. **个性化定制**：基于详细问卷生成独特人设
4. **视觉化呈现**：结合图片和文字的综合展示
5. **用户友好**：简洁直观的操作界面
